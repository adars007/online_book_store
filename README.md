# online_book_store
This project is an Online Book Store data analysis system developed using Microsoft Excel and SQL. The main objective of this project is to manage, analyze, and extract meaningful insights from book sales data, customer information, and order transactions.
Using SQL, the project focuses on querying relational databases to perform operations such as filtering, aggregation, joins, and subqueries for business insights. Excel is used for data cleaning, formatting, pivot tables, and basic visualization to support decision-making.
